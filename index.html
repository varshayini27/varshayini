Pet Rescue Centralized Platform

Technical Challenge Building a multi-tenant resource sharing model — where each center retains data ownership yet allows cross-center visibility — required carefully scoped API authorization roles and a shared-yet-segmented database design.

A hardware project bridging embedded systems and user experience — a fully automated coffee machine that reads user preferences (strength, size, sugar) and executes precise brewing sequences without manual intervention.
